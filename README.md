# 📚 Sistema de Livraria Simples

> Sistema em Java para gerenciamento completo de livros, clientes, locações e controle de estoque.

---

## 🛠️ Funcionalidades do Sistema
* **👥 Cadastro de Clientes & Usuários:** Controle completo de quem frequenta a livraria.
* **📖 Catálogo de Livros:** Separação organizada entre Livros Técnicos e Livros de Literatura.
* **🔄 Sistema de Locação:** Controle de empréstimos com atualização de status.
* **🔐 Tela de Login:** Acesso seguro para gerenciamento do sistema.
